# 🏍️ Moto-QR Chile

**Sistema de Identificación de Emergencia de Riders — Para Santos y Pecadores**

Sistema web para el registro de motociclistas y la generación de stickers QR de emergencia. Al escanear el código QR del casco, los equipos de rescate (SAMU, Bomberos, Carabineros) acceden instantáneamente a la información médica crítica del rider.

## 🚀 Sitio en vivo
https://mchd-cl.github.io/MotoQR/

## ✨ Funcionalidades
- Registro de riders con datos médicos de emergencia
- RUT chileno con formato y validación de duplicados
- Fecha de nacimiento con cálculo automático de edad
- Región y comuna (todas las comunas de Chile)
- Generación de sticker QR formato 8×6 cm
- QR que abre pantalla de emergencia con grupo sanguíneo, alergias y contacto
- Botón WhatsApp directo al contacto de emergencia
- Panel de administrador protegido con código
- Exportación a Excel
- Sincronización en tiempo real (Firebase)
- Copia automática a Google Sheets
